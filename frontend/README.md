# InfraExplain Frontend

A modern React + Vite frontend for the InfraExplain Terraform code explanation service.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Configuration

The API URL can be configured via environment variable:

- Create a `.env` file with `VITE_API_URL=http://localhost:8080`
- Or the default is `http://localhost:8080`
